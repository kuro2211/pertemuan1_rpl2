**RPL Pertemuan 1 – Dasar Pemrograman Java**

Repository ini berisi contoh program Java yang digunakan untuk mempelajari konsep dasar pemrograman pada pertemuan pertama mata kuliah RPL (Rekayasa Perangkat Lunak).

**Deskripsi Program**

1. Program ini menampilkan penerapan beberapa konsep dasar Java, seperti:

2. Deklarasi variabel

3. Operasi aritmetika

4. Input dari pengguna (Scanner)

5. Perulangan (for)

6. Percabangan (if – else if – else)
